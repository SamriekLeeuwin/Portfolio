BudgetLife

BudgetLife is a full-stack budgeting application currently under development. It allows users to manage their income and expenses efficiently, set saving goals, and track their financial health over time. The project uses Node.js, Express.js, MySQL for the backend, and modern web technologies (HTML, CSS, JavaScript) for the frontend. The app follows RESTful principles and includes authentication features to securely manage user data.
